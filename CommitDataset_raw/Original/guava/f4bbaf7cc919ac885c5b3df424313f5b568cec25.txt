Adds an addAll(Stats) method to add a Stats object to a StatsAccumulator, and corresponding methods for PairedStats and PairedStatsAccumulator.

This is useful for merging statistics accumulated by multiple StatsAccumulators. One application is using a saved Stats object to restore the state of a StatsAccumulator.
