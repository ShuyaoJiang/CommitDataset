Allow a few build-time arguments to the C++ build

Add DEBUG and PIC flags to scons so you can now build without debug, e.g.,
  scons DEBUG=false lib
or with -fPIC forced, e.g,.
  scons PIC=true lib

Defaults haven't changed.
